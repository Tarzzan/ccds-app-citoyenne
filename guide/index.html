Installer Expo Go

L'application de test sur votre smartphone

▼
💡 Expo Go vous permet de tester l'application CCDS directement sur votre téléphone, sans avoir à la compiler ou à passer par les stores.
🤖
Disponible sur Google Play
🍎
Télécharger sur App Store
⚠️ Sur iOS, vous pouvez aussi simplement ouvrir l'appareil photo natif et scanner le QR Code — il ouvrira automatiquement Expo Go.
2

Lancer le serveur de développement

Démarrer Expo sur votre ordinateur

▼

Sur votre ordinateur, ouvrez un terminal dans le dossier du projet et exécutez les commandes suivantes :

cd ccds-app-citoyenne/mobile
npx expo start
📱 Expo affichera un QR Code dans le terminal. Scannez-le avec Expo Go (Android) ou l'appareil photo (iOS) pour ouvrir l'application sur votre téléphone.
3
